# US속보 웹 개발 진행 로그

## Phase 1 — 기반 인프라 (완료)
- 극장 좌석 100석 + 캐릭터 커스텀 (후드색, 눈, 머리, 이니셜)
- Supabase Auth + Google OAuth
- 마이페이지 프로필 카드 (레벨/XP/포인트)
- 해지 경고 모달

### 오류/이슈
- 착석 시 `insert` 사용 → 기존 subscribers 행과 충돌 → `upsert`로 수정 (3711cbd)
- payment_status: "test"로 저장 → isSubscriber 인식 안됨 → "active"로 수정 (ae7f730)

## Phase 2 — 커뮤니티 + 경험치 (완료)
- `/community` 페이지 MVP (글 + 댓글, 구독자만 작성)
- 경험치(XP) + 포인트 자동 적립 시스템 (Supabase RPC grant_xp)
- XP 보상: 뉴스 +1, 카드뉴스 +2, 글 +5, 댓글 +2, 피드백 채택 +20, 재구독 +50
- Navbar 커뮤니티 링크 추가

### 오류/이슈
- `post_likes` 테이블 생성 필요 → Supabase에서 직접 생성
- 커뮤니티 글쓰기 불가 → payment_status 문제와 동일 원인

## Phase 2.5 — 캐릭터 + UI 개선 (완료)
- 캐릭터 디자인 개선: 둥글고 귀여운 몸통/팔/발
- 눈 6종 (+윙크, 반짝이), 머리 6종 (+뾰족, 단발)
- 피부색 4종 (라이트~다크)
- 시즌1 테두리 3종 — 골드/다이아/불꽃 (좌석 박스 border)
- 액세서리 5종 — 선글라스, 안경, 에비에이터, 모노클
- 커뮤니티 따봉(추천) + 핫게시글(3추천+1주일) + HOT 뱃지
- AuthorBadge에 유저 캐릭터 SVG 표시

### 오류/이슈
- 테두리가 캐릭터 머리 주변 원형으로 렌더링 → 좌석 박스 border로 변경 (0a4965a)
- 캐릭터 작게 보임 → size 66→78로 확대
